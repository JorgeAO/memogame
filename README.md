# MemoGame 🧩

¡Un divertido juego de memoria diseñado para niños!

## Cómo jugar
1. Elige tu tema favorito (Animales, Juguetes o Espacio).
2. Selecciona la dificultad (Fácil, Medio o Difícil).
3. Voltea dos fichas buscando que tengan la misma figura.
4. Si coinciden, ¡desaparecen mágicamente! 🌟
5. Si no coinciden, se darán la vuelta para que vuelvas a intentar.
6. ¡Gana el juego encontrando todos los pares en el menor tiempo posible!

## Características principales
- **Diseño Móvil**: Optimizado para jugar en celulares y tablets.
- **Efectos 3D**: Las cartas giran suavemente al tocarlas.
- **Temas infantiles**: Emojis coloridos y amigables.
- **Seguimiento de progreso**: Temporizador y contador de movimientos integrados.
- **Celebración final**: ¡Confetti de emojis al ganar!

## Cómo ejecutar localmente
Si tienes Node.js instalado, puedes usar:
```bash
npx serve .
```
O simplemente abre el archivo `index.html` en tu navegador preferido.

## Tecnologías utilizadas
- HTML5 Semántico
- CSS3 (Flexbox/Grid/3D Transforms)
- Vanilla Javascript (ES6+)
- Google Fonts (Outfit)
